# System Vulnerability Checklist
DecodeLabs Cyber Security Internship - Project 4 (Batch 2026)

## Goal
Perform a proactive security audit of a Windows system and rank the identified risks.

## Audit Summary
Six security checks were performed using built-in Windows tools (PowerShell and Settings): firewall status, disk encryption, patch status, login authentication, screen lock, and MFA. 3 of 6 checks passed. The failures are risk-ranked (2 High, 1 Medium) with a prioritized remediation plan.

## Key Skills
Risk assessment, system hardening, defensive logic
